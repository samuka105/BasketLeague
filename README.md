# BasketLeague
## configuracao do App:
na pasta /front:
    
    yarn
    
para rodar:
    
    expo start:web 
    
## configuracao da API:
na pasta /api:
    
    npm install
para rodar:

    node index.js
### Exemplos de chamada pelo browser:
* localhost:3001/player?name=Stephen Curry
* localhost:3001/player?id=201939
